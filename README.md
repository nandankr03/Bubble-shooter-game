# 🎮 Bubble Shooter Game

A modern Bubble Shooter game inspired by the classic arcade game.

## Features
- 🎯 Classic Bubble Shooter Gameplay
- 🌈 Colorful Bubble Animations
- 📊 Score & High Score System
- ⏸ Pause & Resume
- 🔄 Restart Game
- 📱 Responsive Design
- ⚡ Smooth Canvas Animations
- 💻 Built with HTML5 Canvas, CSS3 and Vanilla JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript
- HTML5 Canvas
